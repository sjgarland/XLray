About Yeoman Guide

Yeoman Guide demonstrates how to use the Yeoman Office generator successfully to develop and deploy an add-in for Microsoft 365 Excel. It shows how to overcome problems with the Angular/Typescript prototype generated by Yeoman.

Yeoman Guide is an open source project created by Stage One Software, the developers of the Formula Forge add-in for Excel.
